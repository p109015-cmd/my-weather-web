## 📝 專案簡介 (Project Overview)
本專案是由 **賴以航** 獨立開發的雲端全端自動化 AI 天氣問答助理。系統採用 **Python 3.14** 尖端雲端環境部署，完美串接即時氣象大數據 API (`wttr.in`)。

使用者只需輸入台灣任意縣市名稱，系統即可即時進行大數據分析，提供直觀、可視化的智慧氣象儀表板，並針對當前氣候給出 AI 穿搭與防雨建議。

---

## ✨ 核心核心功能 (Key Features)
* **🌍 全台灣縣市支援：** 內建 20 個台灣主要縣市名稱智慧對照字典。
* **📊 智慧大數據儀表板：** 一眼掌控「目前氣溫」、「空氣濕度」與「即時降雨機率」。
* **🔤 超級天氣狀態中文化：** 獨家實裝高達數十種氣象狀態的「全小寫防錯中文字典」，徹底解決傳統網頁文字過長被截斷與語系衝突問題。
* **💡 AI 智慧穿搭與防雨決策：** 系統自動根據降雨機率與氣溫，產出人性化的出門帶傘提示與穿著衣物建議。
* **📈 未來三日氣象趨勢圖：** 動態生成未來三天最高與最低氣溫折線圖，掌握天氣走向。
